
## Quiz Example

Ejemplo de juego de preguntas y respuestas desarrollado en laravel

## Install locally
### Linux
```shell
    cp ./build/docker-compose.yml.dist ./docker-compose.yml
    docker-compose up -d --build
```
### Windows
```shell
    mv ./build/docker-compose.yml.dist ./docker-compose.yml
    docker-compose up -d --build
```
